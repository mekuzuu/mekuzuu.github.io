Alacrittyを使い始めた

January 4, 2022

ターミナルエミュレータをAlacrittyに変えてみました。

Alacritty

Rust製のターミナルエミュレータです。

GPUでレンダリングするため高速らしいです。

https://github.com/alacritty/alacritty

モチベーション

なんとなくiTerm2を使っていたが、最近Rust製のツールが便利で色々試していたところ、Alacrittyを知りました。

高速かつyamlで設定ができる点に魅力を感じました。

インストール

Homebrewを使っているので、下記コマンドで入れました。

brew install alacritty

Configuration

設定は以下を参考にしました。

https://github.com/chocoby/dotfiles/blob/master/.alacritty.yml

Font

font:
  normal:
    family: JetBrains Mono
    style: Regular

Shell

fish shellを起動するように設定。

shell:
  program: /usr/local/bin/fish

KeyBindings

key_bindings:
  - { key: Key0,         mods: Command,           action: ResetFontSize }
  - { key: Semicolon,    mods: Command|Shift,     action: IncreaseFontSize }
  - { key: Minus,        mods: Command|Shift,     action: DecreaseFontSize }
  - { key: N,            mods: Command,           action: SpawnNewInstance }

感想

いまのところキビキビ動いています。

高速が故にミニマムな設計ですが、慣れれば問題ない印象です。

しばらくは Alacritty を使ってみようと思います。
