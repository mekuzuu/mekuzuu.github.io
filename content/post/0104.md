ターミナルエミュレータをiTerm2からAlacrittyに変えてみた。

## Alacritty

Rust製のターミナルエミュレータ。

GPUでレンダリングするため高速らしい。

[https://github.com/alacritty/alacritty](https://github.com/alacritty/alacritty)

## モチベーション

なんとなくiTerm2を使っていたが、最近Rust製のツールが便利で色々試していたところ、Alacrittyを知った。

高速かつyamlで設定ができる点に魅力を感じた。

## インストール

`Homebrew`を使っているので、下記コマンドで入れた。

```sh
brew install alacritty
```

## Configuration
設定は以下を参考にした。

[https://github.com/chocoby/dotfiles/blob/master/.alacritty.yml](https://github.com/chocoby/dotfiles/blob/master/.alacritty.yml)

## Font
```yaml
font:
  normal:
    family: JetBrains Mono
    style: Regular
```

## Shell
fish shellを起動するように設定した。
```yaml
shell:
  program: /usr/local/bin/fish
```

## KeyBindings
```yaml
key_bindings:
  - { key: Key0,         mods: Command,           action: ResetFontSize }
  - { key: Semicolon,    mods: Command|Shift,     action: IncreaseFontSize }
  - { key: Minus,        mods: Command|Shift,     action: DecreaseFontSize }
  - { key: N,            mods: Command,           action: SpawnNewInstance }
```

## 感想

iTerm2よりも高速に動作している。

高速が故にミニマムな設計だが、慣れれば問題ない印象。

しばらくは Alacritty を使ってみる。
